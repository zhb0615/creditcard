## 用卡策略
- **火种卡原则**：每家银行至少保留1张卡。银行唯一卡不得注销（即使该卡回报率低或无专属权益）。工行、中行、平安均属此类。
- **建行火种卡**：全球支付VISA·双币种（Visa+银联），正青春和欢享可销。
- **当前精简目标**：18张 → 14张（注销：天天利臻享版+正青春+欢享+农行车主尊享版）
